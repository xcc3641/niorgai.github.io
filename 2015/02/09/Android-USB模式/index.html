Android USB模式

官方文档提供的USB Host and Accessory说明中，说到USB模式包括Host模式和Assessory模式，介绍说明为： When the Android-powered device is in host mode, it acts as the USB host and powers the bus. When the Android-powered device is in USB accessory mode, the connected USB hardware (an Android USB accessory in this case) acts as the host and powers the bus.

由图可以说明：

  • Assessory模式
    • 类似于U盘模式，USB设备驱动，电脑操控手机，理论上所有手机都支持。
    • 支持Android 3.1(level 12)以上版本。可以通过添加依赖包支持Android 2.3.4(level 10).
  • Host模式
    • Android设备作为Host，可以外接鼠标、手柄等。
    • USB硬件必须支持OTG，如U盘不支持。
    • Android系统必须3.1以上，并且有OTG驱动。
